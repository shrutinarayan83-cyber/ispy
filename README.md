# 🧭 ISpy: Mumbai Explorer

> *Every ride an adventure.*

A real-world GPS landmark discovery game for Mumbai. Turn every journey into a discovery — spot real landmarks, answer quizzes, earn points.

---

## 🎮 How It Works

1. Open the app and allow location access
2. Pick a route
3. As you travel, the app detects when you're near a landmark and unlocks it
4. Answer the quiz, learn a fun fact, earn points
5. Complete all spots on the route to finish

---

*Built with curiosity, paper sketches, and lots of real-world testing on Mumbai roads. 🛣️
